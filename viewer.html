Figma UI components (10 sections)

아래는 Figma 파일 Seeds-The-Day-Project---Seeday 의 10개 섹션 노드에 맞춘 HTML/CSS 구현입니다. 각 섹션은 components/*-section.js의 mount*Section으로 마운트되며, variant 키는 *_VARIANTS에 정리되어 있습니다. 각 섹션 제목 바로 아래 구현 소스에서 .js·.css 파일을 새 탭으로 열거나, 이 페이지에서 보기를 펼치면 같은 화면에서 코드를 불러옵니다(file:// 로 열면 fetch가 막힐 수 있어 HTTP 서버 사용을 권장). 대부분의 미리보기는 클릭하면 다음 상태로 순환합니다(포커스 후 Enter·Space 동일). 드롭다운 필터는 칩을 직접 눌러 활성 탭을 바꿀 수 있습니다. 터미널에서는 프로젝트 루트에서 ./scripts/open.sh 또는 ./scripts/open-viewer.sh 를 실행하면 서버가 뜨고 이 페이지가 열립니다. 토큰 미리보기는 ./scripts/open.sh tokens 또는 ./scripts/open-tokens-preview.sh. 포트 변경: PORT=9000 ./scripts/open.sh. 수동으로 띄우려면 python3 -m http.server 8877 후 /viewer.html 로 접속하면 됩니다.
